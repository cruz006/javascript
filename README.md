## How to upload to GitHub

To make a new repository, do this:

```bash
git init
```

then you can type this in

```bash
git add .
```

now write a local commit with a message:

```bash
git commit -m 'YOUR MESSAGE'
```

once this is finished, push your local commit:

```bash
git push
```
